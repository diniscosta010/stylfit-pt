# Instruções de Instalação - Stylfit PT

## 1️⃣ Instalar Node.js

Antes de começar, você precisa instalar o Node.js:

1. Visite [https://nodejs.org/](https://nodejs.org/)
2. Faça download da versão LTS (recomendada)
3. Execute o instalador e siga as instruções
4. Reinicie seu computador

Para verificar se foi instalado corretamente, abra o PowerShell e execute:
```
node --version
npm --version
```

## 2️⃣ Instalar Dependências

Navegue até a pasta do projeto e execute:
```
npm install
```

## 3️⃣ Executar em Desenvolvimento

```
npm run dev
```

Acesse: http://localhost:3000

## 4️⃣ Build para Produção

```
npm run build
npm start
```
